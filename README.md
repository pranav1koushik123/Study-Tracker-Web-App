# 📚 Study Tracker App

A web application that helps students track their daily study sessions, subjects, and study hours.  
This project allows users to manage subjects, record study sessions, and view study statistics on a dashboard.

The goal of this project is to practice **backend development using Flask and databases**.

---

## 🚀 Features

- User Registration and Login
- Add and manage subjects
- Record study sessions
- Track hours studied
- Dashboard with study statistics
- Data visualization using Chart.js
- Secure password hashing
- Simple and clean UI using Bootstrap

---

## 🛠 Tech Stack

### Frontend
- HTML
- CSS
- Bootstrap
- JavaScript
- Chart.js

### Backend
- Python
- Flask

### Database
- MySQL
- SQLAlchemy ORM
